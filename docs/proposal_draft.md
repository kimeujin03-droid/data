# 컬처갭 AI: 공공 문화데이터 기반 문화 사각지대 예측지도
## 제4회 문화체육관광 인공지능·데이터 활용 공모전 기획서

---

## 1. 제안 배경

### 1.1 문화 접근성 격차 문제

우리나라의 문화시설 수와 문화행사 수는 꾸준히 증가하고 있다.
그러나 이 성장의 혜택이 모든 지역 주민에게 고르게 돌아가는지는 별개의 문제다.

실제로 어떤 지역에 문화시설이 존재하더라도 다음과 같은 이유로 주민이 문화를 향유하지 못하는 경우가 있다.

- 시설은 있지만 공연·전시·체험 프로그램이 거의 열리지 않는다
- 고령층 비율이 높은 지역인데 야간 공연 위주로 행사가 구성되어 있다
- 청소년 인구가 많은 지역인데 교육·체험형 프로그램이 부족하다
- 읍면 지역은 가까운 문화시설까지 거리가 멀고 대중교통이 취약하다
- 무료 또는 저비용 프로그램이 부족하여 경제적 접근성이 낮다

이처럼 문화 사각지대는 단순히 "시설이 없는 곳"이 아니다.
**문화 수요가 존재하지만 시설·행사·교통·가격·프로그램 적합성 중 하나 이상이 충족되지 못하는 지역**으로 정의해야 한다.

### 1.2 시설 수 중심 분석의 한계

기존 문화격차 분석은 주로 지역별 문화시설 수, 인구 대비 시설 수, 특정 시설 접근거리를 비교하는 데 그쳤다.
그러나 이런 방식은 실제 문화 향유의 어려움을 설명하지 못한다.

시설이 있어도 프로그램이 없거나, 가격이 비싸거나, 대상이 맞지 않으면 그 지역 주민에게 그 시설은 사실상 없는 것과 다름없다.

### 1.3 AI·데이터 기반 문화 사각지대 진단 필요성

공공 문화데이터, 인구 데이터, 교통 데이터를 결합하면 실질적인 문화 접근성 결핍을 정량화할 수 있다.
특히 AI를 활용한 행사 텍스트 자동 분류와 지역 유형 군집화를 통해
각 지역에 필요한 문화공급 전략을 구체적으로 제안하는 것이 가능하다.

## 2. 문제 정의

### 2.1 문화 사각지대 정의

> 문화 수요가 존재함에도 불구하고,  
> 시설 공급·행사 공급·접근성·프로그램 적합성 중 하나 이상이 부족하여  
> 주민이 실질적으로 문화를 향유하기 어려운 지역.

### 2.2 핵심 분석 질문

> 문화데이터와 AI를 활용하면,  
> 지역별 문화 접근성 결핍을 정량화하고  
> 어느 지역에 어떤 문화공급이 필요한지 예측할 수 있는가?

### 2.3 프로젝트 목표

1. 전국 247개 시군구 단위의 문화결핍지수를 산출한다.
2. 문화 사각지대 고위험 지역을 지도 형태로 시각화한다.
3. 지역별 결핍 원인을 4가지 유형으로 자동 진단한다.
4. 지역 유형에 따른 맞춤형 문화공급 전략을 제안한다.
5. 지자체와 문화정책 담당자가 예산과 프로그램을 효율적으로 배분할 수 있도록 지원한다.

## 4. 분석 및 AI 방법론

### 4.1 데이터 전처리

- 지역명 정규화: 시·구 병기 형식 통일, 시 총합 행 제거 (성남시 → 분당구/수정구/중원구 분리)
- 행사 기간 파싱: YYYYMMDD 형식 → 날짜 필터링 (오늘 기준 종료 행사 제거: 9,890→1,898건)
- 위경도 정제: 결측치 제거, 경계 외 좌표 제외
- 축제 데이터: xlsx 파싱 → 시군구 매칭 → 동일 스키마 통합

### 4.2 지역별 집계 (시군구 단위)

- 문화시설 수 집계 (시설 유형별)
- 월평균 행사 수, 무료 행사 비율, 장르 다양성 지수
- 버스정류장 수 (인구 1만 명당)
- 연령별 인구 집계: 청소년(10~24세) 비율, 고령층(65세+) 비율

### 4.3 AI 활용 1: 문화행사 텍스트 자동 분류 (1단)

9,890개 행사·축제 제목에 대해 4개 축을 자동 태깅한다.

| 분류 축 | 분류값 예시 | 방법 |
|---|---|---|
| 장르 | 공연/전시/체험·교육/축제/기타 | 이벤트 유형 코드 + 키워드 매핑 |
| 대상계층 | 일반·전문/가족·아동/청소년·교육/고령친화 | 수혜자 명시 키워드 분류 (보수적 적용) |
| 시간대 | 주간/야간/종일 | 야간·축제·페스티벌 키워드 |
| 가격 | 유료/무료·저비용 | 제목 키워드 우선, 장르 기본값 보완 |

**보수적 분류 설계**: "노인의 꿈" 같은 고령층 소재 작품은 고령친화로 잘못 분류되는 것을 방지.
수혜자가 명시된 경우("어르신 대상", "시니어 프로그램")만 고령친화로 태깅.

분류 결과 (1,898건 활성 행사 기준):
- 가격: 유료 약 41% / 무료·저비용 약 59%
- 대상: 일반·전문 76% / 가족·아동 13% / 고령친화 4% / 청소년 7%

### 4.4 문화공급지수 산출

```
Supply Score
= 0.25 × 시설밀도점수(인구 1만명당 시설 수)
+ 0.25 × 행사밀도점수(월평균 행사 수)
+ 0.20 × 프로그램다양성점수(장르 다양성 지수)
+ 0.15 × 무료행사비율점수
+ 0.15 × 상시공급점수(축제 외 상시 프로그램 비율)
```

각 구성 점수는 전국 247개 시군구 min-max 정규화 (0~100).

### 4.5 문화수요지수 산출

```
Demand Score
= 0.30 × 인구규모점수
+ 0.20 × 청소년수요점수(10~24세 비율)
+ 0.20 × 고령층수요점수(65세+ 비율)
+ 0.15 × 농산어촌취약점수(읍면 지역 여부)
+ 0.15 × 교통취약점수(버스정류장 밀도 역수)
```

### 4.6 접근성 패널티

```
Access Penalty
= 0.40 × 거리불리점수(인근 문화시설까지 평균거리)
+ 0.30 × 대중교통취약점수(정류장 밀도 역수)
+ 0.20 × 농산어촌보정
+ 0.10 × 고령층이동취약보정
```

### 4.7 프로그램 미스매치 패널티

지역 인구구조와 실제 문화행사 프로필의 불일치를 측정한다.

예: 고령층 비율이 높은데 야간 유료 공연 위주 → 미스매치 높음

### 4.8 문화결핍지수 산출

```
Culture Gap Score
= 0.40 × Demand Score
+ 0.30 × (100 - Supply Score)
+ 0.20 × Access Penalty
+ 0.10 × Mismatch Penalty
```

**위험 등급 분류:**

| 점수 | 등급 |
|---:|---|
| 80점 이상 | 고위험 |
| 60~79점   | 주의 |
| 40~59점   | 보통 |
| 0~39점    | 양호 |

### 4.9 AI 활용 2: 지역 유형 군집화 (K-Means)

문화결핍 원인에 따라 4가지 유형으로 군집화:

| 유형 | 설명 | 전국 비율 |
|---|---|---:|
| 고령층 교통취약형 | 고령 인구 높고 대중교통 취약, 낮 시간대 무료 프로그램 부족 | 37% |
| 시설·행사 부족형 | 문화시설과 정기 행사 자체가 부족 | 30% |
| 균형 관리형 | 전반적으로 양호하나 특정 계층 공백 | 30% |
| 교통 취약형 | 시설은 있으나 접근성 불리 | 3% |

### 4.10 AI 활용 3: 맞춤형 문화공급 추천 (2단)

지역 유형별 결핍 프로필에 기반해 실제 활성 행사를 매칭 추천한다.

**추천 로직:**
1. 오늘 기준 진행 중인 행사만 대상 (종료 행사 제거)
2. 지역 결핍 프로필과 행사 태그의 일치도로 기본 점수 산출
3. Haversine 거리 계산: 행사 위치 ↔ 시군구 중심점
   - 교통취약형: 10km 이내 +4점 / 100km 초과 -3점 (강력 패널티)
4. 다양성 재랭킹: 같은 행사가 다른 지역에 중복 추천되지 않도록 제어
5. Top-10 행사 추천, culture.go.kr 직접 링크 포함

**추천 결과 품질:**
- 135개 결핍 지역(60점 이상)에 1,350건 추천
- Top-1 행사 고유 비율: 134/135개 지역 (복붙 없음)
- 가격 다양성: 유료 추천 포함 (무료 일변도 방지)

## 7. 기대효과

### 7.1 문화예산 배분 효율화

문화결핍지수와 지역 유형을 기반으로 예산 우선순위를 설정할 수 있다.
"어느 지역에, 어떤 유형의 문화공급이 필요한가"에 대한 데이터 근거를 제공한다.

### 7.2 지역 문화격차 완화

문화 사각지대 지역에 맞춤형 문화서비스를 공급하여 지역 간 문화 향유 격차를 줄인다.

### 7.3 문화소외계층 포용

고령층, 청소년, 농산어촌 주민, 교통취약 지역 주민 등 취약계층에 적합한 프로그램을 우선 추천한다.

### 7.4 지자체 정책 의사결정 지원

정책 시뮬레이션 기능을 통해 "특정 프로그램을 추가했을 때 결핍지수가 얼마나 개선되는가"를
수치로 보여줌으로써 문화 사업 예산 심의와 기획을 지원한다.

## 8. 차별성 및 발전 가능성

### 8.1 기존 문화정보 서비스와의 차별성

기존 서비스는 문화시설 위치, 행사 일정을 검색·제공하는 데 그친다.
컬처갭 AI는 이를 넘어 **문화결핍 원인을 진단하고 정책 처방을 제안하는 정책지원 AI**다.

| 구분 | 기존 서비스 | 컬처갭 AI |
|---|---|---|
| 기능 | 문화정보 검색·제공 | 결핍 원인 진단 + 정책 처방 |
| 주체 | 국민 검색 | 정책 담당자 의사결정 지원 |
| 분석 단위 | 개별 행사·시설 | 지역 단위 수요-공급 불균형 |
| AI 활용 | 검색 추천 | 텍스트 분류 + 군집화 + 예측 |

### 8.2 전국 지자체 확장 가능성

시군구 단위 → 읍면동 단위로 정밀화할 수 있다.
지자체별로 "우리 지역 특화 문화결핍지도"를 제공하는 SaaS형 서비스로 발전 가능하다.

### 8.3 타 부처 연계 가능성

- 보건복지부: 고령층 문화복지 서비스 연계
- 교육부: 청소년 방과후 문화예술 프로그램 지원
- 국토교통부: 문화시설 접근성 기반 지역개발 계획 지원
